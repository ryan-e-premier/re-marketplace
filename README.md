# Claude Marketplace

A collection of Claude Code plugins for code review and development utilities.

## Installation

```bash
/plugin marketplace add ryan-e-premier/claude-marketplace
```

## Plugins

- [`re`](./plugins/re/README.md) — Slash commands for PR review, plan
  review, diffview, transcripts, timecards, and more
- [`diff-review`](./plugins/diff-review/README.md) — Hook-based vimdiff
  review before file changes are applied
